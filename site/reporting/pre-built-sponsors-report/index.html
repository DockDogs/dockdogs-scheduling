Introduction

There are 3 pre-built sponsors reports available already configured inside of Smartsheet. These have pre-defined schedules assigned to them which will send an email to each sponsor on a weekly basis. An example report can be found below.

Selecting A Sponsor

In order to have an event show in the SPONSORSHIP REPORTS you need to designate which sponsor will have their dock at the event. This can be done by using the SPONSOR column inside of Smartsheet.

Confirmed Versus Tentative Status

Only events with the GREEN or YELLOW status fields selected in the POSTED column will be included in these reports. This ensures that sponsors are only notified of events that have been POSTED on the DockDogs website and opened for registration.
